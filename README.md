# Guno_Final

## Clone the Repository

Download the project:

``` bash
git clone https://github.com/nerfeguno/Guno_Final.git
cd Guno_Final
```

## Updating the Project (Best Practice Workflow)

### Always start with `git pull`

``` bash
git pull
```

### Check changes

``` bash
git status
```

### Add updated files

``` bash
git add filename.java
```

Or add all:

``` bash
git add .
```

### Commit with a message

``` bash
git commit -m "Describe what you updated"
```

### Push to GitHub

``` bash
git push
```

## Compiling & Running Java Files

### Compile a single Java file

``` bash
javac FileName.java
```

### Run the compiled file

``` bash
java FileName
```

### Compile all Java files

``` bash
javac *.java
```

### Run the main class

``` bash
java MainClass
```
